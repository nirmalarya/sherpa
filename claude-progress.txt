ðŸ’¡ KEY LEARNINGS
=================
1. Command Palette Best Practices:
   - Use global keyboard listener at top-level component
   - Ctrl+K is industry standard (VSCode, GitHub, etc.)
   - Support both Ctrl (Windows/Linux) and Cmd (Mac)
   - Prevent default browser behavior
   - Clean up event listeners on unmount

2. Keyboard Navigation Implementation:
   - Track selected index with useState
   - Arrow keys modify index within bounds
   - Visual feedback essential (highlight, arrows, colors)
   - Scroll selected item into view
   - Enter executes, Escape closes
   - Mouse hover updates selection for hybrid UX

3. React Modal Best Practices:
   - Portal-style overlay with fixed positioning
   - Semi-transparent backdrop (bg-opacity-50)
   - Click outside to close
   - Keyboard shortcuts for close (Escape)
   - Focus management (auto-focus search input)
   - Clean state reset on close/reopen

4. Command Palette UX:
   - Search filter makes commands discoverable
   - Icons improve visual scanning
   - Descriptions provide context
   - Keyboard hints teach users
   - First item selected by default
   - Smooth animations feel professional

5. Integration Patterns:
   - Use data-attributes for programmatic access
   - setTimeout allows React render before triggering
   - Navigate first, then trigger modal
   - Clean separation of concerns
